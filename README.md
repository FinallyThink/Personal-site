# 项目目录简介
```
|—— public           #html 模板目录    
|—— src              #开发目录    
   |——assets         #项目资源目录    
   |——components     #组件目录    
   |——body           #页面主体组件  
   |——header         #页头组件  
   |——index          #主页  
   |——public         #公共组件/样式  
   |——lib            #模拟数据目录  
      |——config.js   #全局数据注入  
|——APP.vue           #入口文件    
```
